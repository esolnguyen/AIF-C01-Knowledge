### Q1. In a generative AI text pipeline, what is a "token"?
- A. A security credential used to authenticate calls to Amazon Bedrock
- B. A basic unit of text (a word or sub-word) that the model processes
- C. A numeric vector representing the meaning of a document
- D. A single training epoch of a foundation model

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

A token là đơn vị văn bản cơ bản (từ hoặc phần của từ) mà LLM xử lý; billing của nhiều dịch vụ GenAI dựa trên số token vào/ra. Vector meaning là embeddings (C), không phải token.
</details>

### Q2. A team splits long documents into smaller passages before creating embeddings for a search application. What is this technique called?
- A. Tokenization
- B. Fine-tuning
- C. Chunking
- D. Quantization

<details><summary>Đáp án & giải thích</summary>

**Đáp án: C**

Chunking là chia tài liệu dài thành các đoạn nhỏ để embed và lưu trong vector store, giúp truy hồi chính xác hơn (thường dùng trong RAG).
</details>

### Q3. Which statement best describes an embedding?
- A. A dense numeric vector that captures the semantic meaning of text or data
- B. A prompt template used to guide model output
- C. A managed endpoint that hosts a foundation model
- D. A compression format that reduces model file size

<details><summary>Đáp án & giải thích</summary>

**Đáp án: A**

Embeddings là các vector số biểu diễn ngữ nghĩa; các mục có ý nghĩa tương tự sẽ gần nhau trong không gian vector, cho phép tìm kiếm theo độ tương đồng.
</details>

### Q4. Which type of foundation model is specifically designed to generate high-quality images from text prompts?
- A. Transformer-based LLM
- B. Diffusion model
- C. Regression model
- D. Decision tree ensemble

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

Diffusion models (ví dụ Stable Diffusion) tạo ảnh bằng cách khử nhiễu dần từ noise, là kiến trúc phổ biến cho image generation. LLM chủ yếu cho văn bản.
</details>

### Q5. A company wants a single model that can accept both text and images as input and reason over them together. Which characteristic should the model have?
- A. Multi-modal
- B. Nondeterministic
- C. Provisioned throughput
- D. Zero-shot

<details><summary>Đáp án & giải thích</summary>

**Đáp án: A**

Multi-modal models có thể nhận và xử lý nhiều loại dữ liệu (text, image, audio…) cùng lúc. Zero-shot là kỹ thuật prompt, không phải khả năng đa phương thức.
</details>

</details>

### Q7. Which sequence correctly represents stages of the foundation model lifecycle?
- A. Deployment → Data selection → Pre-training → Feedback
- B. Data selection → Model selection → Pre-training → Fine-tuning → Evaluation → Deployment → Feedback
- C. Fine-tuning → Pre-training → Data selection → Deployment

</details>

### Q8. A generative AI chatbot occasionally produces confident but factually incorrect answers. What is this limitation called?
- A. Overfitting
- B. Hallucination
- C. Data drift
- D. Underfitting

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

Hallucination là khi mô hình tạo ra nội dung nghe hợp lý nhưng sai sự thật — một hạn chế đặc trưng của GenAI, cần kiểm chứng (ví dụ dùng RAG, human review).
</details>

### Q9. Giving the same prompt to a generative AI model can return different outputs on different runs. Which limitation does this illustrate?
- A. Interpretability
- B. Nondeterminism
- C. Latency
- D. Bias

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

Nondeterminism nghĩa là cùng một input có thể cho ra output khác nhau, một hạn chế cần lưu ý khi cần kết quả nhất quán, có thể tái lập.
</details>

### Q10. An online retailer deploys a generative AI shopping assistant. Which metric BEST measures whether the assistant increases the percentage of visitors who complete a purchase?

</details>

</details>

### Q15. Which characteristic of generative AI is an ADVANTAGE that allows one foundation model to handle many different tasks (summarizing, drafting, translating) without building a separate model for each?

### Q30. An enterprise is deciding between using a foundation model as-is via prompting versus building a custom model. Which statement about the cost tradeoff is MOST accurate?
- A. Custom models always cost less because you avoid all inference charges
- B. Custom (or fine-tuned) models can improve task fit but add data preparation, training, and maintenance costs compared with using a base model via prompting
- C. Using a base model via prompting requires more upfront training cost than a custom model
- D. Both options have identical cost and effort

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

Custom/fine-tuned models có thể cải thiện độ phù hợp tác vụ nhưng phát sinh thêm chi phí chuẩn bị dữ liệu, huấn luyện và bảo trì so với chỉ dùng base model qua prompting. Đây là cân nhắc chi phí then chốt khi chọn hướng customization.
</details>
