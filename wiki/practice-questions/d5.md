# Practice Questions — Domain 5: Security, Compliance & Governance

Exam-style practice for Domain 5 (14%) covering Task Statements 5.1 (secure AI systems) and 5.2 (governance & compliance). Focus on distinguishing similar AWS services.

### Q1. A company runs a generative AI application on Amazon Bedrock and wants to ensure that traffic between its Amazon VPC and Bedrock does not traverse the public internet. Which AWS service should the company use?
- A. AWS PrivateLink
- B. Amazon Macie
- C. AWS Key Management Service (AWS KMS)
- D. AWS WAF

<details><summary>Đáp án & giải thích</summary>

**Đáp án: A**

AWS PrivateLink creates a private VPC interface endpoint so traffic to services like Amazon Bedrock stays on the AWS network and never crosses the public internet. Macie is for data discovery, KMS is for encryption keys, and WAF filters web traffic.
</details>

### Q2. A data science team stores training data in Amazon S3 and needs to automatically discover and classify sensitive data such as personally identifiable information (PII) before using it to fine-tune a model. Which service is designed for this task?
- A. Amazon Inspector
- B. Amazon Macie
- C. AWS Config
- D. Amazon GuardDuty

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

Amazon Macie uses ML to discover, classify, and protect sensitive data (like PII) stored in Amazon S3. Inspector scans workloads for software vulnerabilities, while Config tracks resource configuration state — neither classifies data content.
</details>

### Q3. Under the AWS Shared Responsibility Model, which of the following is the CUSTOMER responsible for when using Amazon Bedrock?
- A. Patching the physical hardware in AWS data centers
- B. Configuring IAM policies that control who can invoke the model
- C. Maintaining the availability of the AWS Global Infrastructure
- D. Securing the hypervisor of the underlying compute

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

AWS secures the underlying infrastructure ("security of the cloud"), while the customer is responsible for "security in the cloud" — including IAM policies, data, and access control. Hardware, global infrastructure, and hypervisor security are all AWS responsibilities.
</details>

### Q4. A compliance officer needs to record and evaluate whether S3 buckets holding AI training data are configured to be encrypted, and be alerted when a bucket becomes non-compliant with that rule. Which service should be used?
- A. AWS CloudTrail
- B. AWS Config
- C. AWS Audit Manager
- D. AWS Artifact

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

AWS Config continuously records resource configurations and evaluates them against rules (e.g., "buckets must be encrypted"), flagging non-compliant resources. CloudTrail records API activity (who did what), not configuration compliance state.
</details>

### Q5. A security team must answer the audit question "Which user invoked the Bedrock model and deleted the guardrail last Tuesday?" Which AWS service provides this record of API calls and account activity?
- A. AWS Config
- B. Amazon Inspector
- C. AWS CloudTrail
- D. AWS Trusted Advisor

<details><summary>Đáp án & giải thích</summary>

**Đáp án: C**

AWS CloudTrail logs API calls and user activity across the account — who did what, when, and from where — making it the tool for auditing specific actions. Config tracks configuration state, not the identity behind each API call.
</details>

### Q6. An organization needs to download AWS's SOC 2 and ISO/IEC 27001 compliance reports to share with an external auditor. Which service provides on-demand access to these AWS audit artifacts?
- A. AWS Audit Manager
- B. AWS Artifact
- C. AWS Config
- D. AWS Trusted Advisor

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

AWS Artifact is the self-service portal for downloading AWS's own compliance reports and agreements (SOC, ISO, PCI, etc.). Audit Manager, by contrast, helps you continuously assess and produce evidence for your OWN workloads' compliance — it does not host AWS's certifications.
</details>

### Q7. A team wants to continuously collect evidence and map it to a framework (such as SOC 2 or GDPR) to streamline audits of their own AI workloads. Which service is purpose-built for automating this evidence collection?
- A. AWS Artifact
- B. AWS Audit Manager
- C. AWS CloudTrail
- D. Amazon Macie

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

AWS Audit Manager automates evidence collection and continuously assesses your resources against prebuilt or custom compliance frameworks. Remember the pairing: Artifact = AWS's reports for you to download; Audit Manager = evidence about YOUR environment.
</details>

### Q8. A company is documenting the intended uses, training details, and limitations of a machine learning model to support transparency and data lineage. Which AWS feature is designed for this documentation?
- A. AWS CloudTrail logs
- B. Amazon SageMaker Model Cards
- C. AWS Glue Data Catalog
- D. Amazon Macie findings

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

Amazon SageMaker Model Cards centralize documentation of a model's intended use, training data, performance, and limitations — supporting source citation, transparency, and governance. A Data Catalog documents datasets, not the model itself.
</details>

### Q9. A generative AI chatbot is vulnerable to an attacker crafting inputs that override the system instructions and cause it to leak confidential data. This attack is known as prompt injection. Which TWO practices help mitigate this risk? (Choose TWO.)
- A. Apply Guardrails for Amazon Bedrock to filter and validate inputs and outputs
- B. Grant the application's IAM role full administrator permissions for flexibility
- C. Store all prompts unencrypted to simplify debugging
- D. Apply input validation and least-privilege access to backend resources the model can reach
- E. Disable all logging to reduce attack surface

<details><summary>Đáp án & giải thích</summary>

**Đáp án: A, D**

Guardrails filter malicious/unwanted prompts and responses, and combining input validation with least-privilege access limits what a successful injection can reach or exfiltrate. Full admin rights, unencrypted storage, and disabling logging all increase risk.
</details>

### Q10. A governance team is classifying its generative AI use cases — from using a public consumer app, to building on a pre-trained foundation model, to training a model from scratch — so it can apply the right security controls to each tier. Which framework is designed to guide this classification?
- A. AWS Well-Architected Framework only
- B. The Generative AI Security Scoping Matrix
- C. ISO/IEC 42001 certification checklist
- D. The AWS Shared Responsibility Model diagram

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

The Generative AI Security Scoping Matrix defines scopes (from consumer apps to self-trained models) and the security considerations for each, helping teams apply appropriate controls. ISO/IEC 42001 is an AI management-system standard, not a scoping tool.
</details>

### Q11. A team needs to manage the encryption keys used to encrypt AI training data at rest in Amazon S3, including controlling key rotation and defining which IAM principals can use each key. Which service provides this capability?
- A. AWS Secrets Manager
- B. AWS Key Management Service (AWS KMS)
- C. Amazon Macie
- D. AWS Certificate Manager (ACM)

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

AWS KMS creates and manages cryptographic keys and controls their use across services like S3, including rotation and key policies. Secrets Manager stores credentials/secrets (not the encryption keys themselves), and Macie classifies data rather than managing keys.
</details>

### Q12. A security team wants to scan the Amazon EC2 instances and container images hosting an AI inference application for software vulnerabilities and unintended network exposure. Which service is purpose-built for this?
- A. Amazon Macie
- B. Amazon Inspector
- C. AWS Config
- D. AWS CloudTrail

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

Amazon Inspector performs automated vulnerability management, scanning EC2 instances and container images for CVEs and unintended network exposure. Macie classifies sensitive data, Config tracks configuration state, and CloudTrail logs API activity — none scan for software vulnerabilities.
</details>

### Q13. A company wants automated recommendations to improve security, cost, performance, and service limits across its AWS account hosting AI workloads — for example, flagging publicly accessible S3 buckets. Which service provides these best-practice checks?
- A. AWS Trusted Advisor
- B. AWS Audit Manager
- C. AWS Artifact
- D. Amazon Inspector

<details><summary>Đáp án & giải thích</summary>

AWS Trusted Advisor inspects your environment and gives best-practice recommendations across cost optimization, performance, security, fault tolerance, and service limits. Audit Manager collects compliance evidence and Artifact hosts AWS's reports — neither provides these general best-practice checks.
</details>

### Q14. Match the intent to the correct service: a team must (1) prove to a regulator that AWS itself holds an ISO/IEC 27001 certification, and (2) continuously gather evidence that THEIR OWN Bedrock workload meets a compliance framework. Which pairing is correct?
- A. (1) Audit Manager, (2) Artifact
- B. (1) Artifact, (2) Audit Manager
- C. (1) Config, (2) CloudTrail
- D. (1) Trusted Advisor, (2) Inspector

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

AWS Artifact provides AWS's own audit reports and certifications (ISO, SOC, PCI) on demand, while AWS Audit Manager continuously collects evidence about YOUR workloads against frameworks. Remember: Artifact = AWS's proof for you; Audit Manager = your proof about you.
</details>

### Q15. Which international standard is specifically an Artificial Intelligence Management System (AIMS) standard that an organization can be certified against to govern its AI practices?
- A. SOC 2 Type II
- B. ISO/IEC 42001
- C. PCI DSS
- D. ISO/IEC 27001

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

ISO/IEC 42001 is the AI management system (AIMS) standard for responsibly governing AI. SOC 2 reports on service-organization controls, ISO/IEC 27001 covers information security management, and PCI DSS covers payment card data — none is AI-specific.
</details>

### Q16. A regulated bank must ensure that customer data used to fine-tune a model never leaves a specific AWS Region and is deleted after 90 days. Which TWO data governance strategies directly address these requirements? (Choose TWO.)
- A. Enforce data residency by selecting the Region and using policies that restrict cross-Region movement
- B. Grant every data scientist administrator access to speed up experiments
- C. Configure S3 lifecycle retention rules to expire the data after 90 days
- D. Disable CloudTrail logging in that Region to reduce noise
- E. Store the data in a single unencrypted bucket for easy access

<details><summary>Đáp án & giải thích</summary>

**Đáp án: A, C**

Data residency controls keep data within a chosen Region, and S3 lifecycle/retention rules automatically expire data to meet the 90-day requirement. Broad admin access, disabling audit logging, and removing encryption all weaken governance and security.
</details>

### Q17. A governance lead must decide between two services: one records "who made each API call" for forensic audits, and another records "what the resource configuration is and whether it complies with rules." Which pairing correctly maps these needs?
- A. Who-called = AWS Config; compliance-state = AWS CloudTrail
- B. Who-called = AWS CloudTrail; compliance-state = AWS Config
- C. Who-called = Amazon Macie; compliance-state = Amazon Inspector
- D. Who-called = AWS Artifact; compliance-state = AWS Audit Manager

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

AWS CloudTrail answers "who did what, when" (API activity), while AWS Config answers "what does the configuration look like and is it compliant" (resource state over time). This is a frequently tested disambiguation.
</details>

### Q18. Scenario: A healthcare startup builds a chatbot on Amazon Bedrock. It must (a) keep API traffic off the public internet, (b) classify PHI in its S3 training bucket, and (c) prevent the model from returning toxic or sensitive content. Which combination of features BEST meets all three needs, in order?
- A. AWS WAF; Amazon Inspector; SageMaker Model Cards
- B. AWS PrivateLink; Amazon Macie; Guardrails for Amazon Bedrock
- C. AWS KMS; AWS Config; CloudTrail
- D. Amazon GuardDuty; AWS Artifact; Trusted Advisor

<details><summary>Đáp án & giải thích</summary>

PrivateLink keeps Bedrock traffic on the private AWS network, Macie discovers and classifies PHI/PII in S3, and Guardrails for Amazon Bedrock filters harmful or sensitive model inputs/outputs. The other options mismatch the tools to the requirements.
</details>

### Q19. To support source citation and data lineage, a team needs to catalog its datasets — recording schema, source, and location — so downstream users can trace where training data originated. Which AWS service is best suited for cataloging the data itself?
- A. Amazon SageMaker Model Cards
- B. AWS Glue Data Catalog
- C. AWS CloudTrail
- D. Amazon Macie

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

The AWS Glue Data Catalog is a central metadata repository that documents datasets (schema, source, location), supporting data lineage and cataloging. Model Cards document the model, not the datasets, and CloudTrail/Macie serve auditing and classification respectively.
</details>

### Q20. Scenario: During an assessment, a team realizes a generative AI feature lets users submit free text that is passed directly into a system prompt with a backend role that has broad database permissions. An attacker crafts input to make the model run unintended queries. Which set of mitigations BEST reduces this prompt-injection risk?
- A. Increase the model's temperature and remove input length limits
- B. Apply Guardrails, validate/sanitize inputs, and apply least-privilege IAM to the backend role
- C. Give the backend role admin access and log everything to compensate
- D. Store prompts in plaintext and disable encryption for faster debugging

<details><summary>Đáp án & giải thích</summary>

**Đáp án: B**

Layered defenses — Guardrails to filter malicious content, input validation/sanitization, and least-privilege IAM so a successful injection can reach little — best mitigate prompt injection. Higher temperature, admin access, and disabling encryption all increase risk rather than reduce it.
</details>
