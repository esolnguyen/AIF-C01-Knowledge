### Q1. A company wants to score millions of accumulated transaction records overnight, where results are not needed until the next morning. Which type of inferencing best fits this requirement?
- A. Real-time inference
- B. Batch inference
- C. Streaming inference
- D. Serverless inference

<details><summary>Answer & explanation</summary>

**Answer: B**

Batch inference processes a large amount of accumulated data at once when immediate results are not needed, optimizing cost. Real-time (A) is used when low-latency responses per request are needed; streaming (C) is not a standard inferencing type on the exam; "serverless" (D) is an infrastructure type, not an inference type.
</details>

### Q2. Which statement correctly describes the relationship between AI, machine learning (ML), and deep learning (DL)?
- A. AI is a subset of ML, and ML is a subset of DL.
- B. Deep learning is a subset of ML, and ML is a subset of AI.
- C. AI, ML, and DL are three independent, unrelated fields.
- D. Machine learning is a subset of deep learning.

<details><summary>Answer & explanation</summary>

**Answer: B**

Nested relationship: AI is the broadest, ML sits within AI, and DL (using multi-layer neural networks) is a subset of ML. The other options reverse the nesting order.
</details>

### Q3. A dataset contains rows and columns where each column represents a customer attribute such as age, income, and region. What type of data is this?
- A. Time-series data
- B. Tabular (structured) data
- C. Image data
- D. Unstructured text data

<details><summary>Answer & explanation</summary>

**Answer: B**

Data in rows-and-columns form with well-defined attributes is tabular / structured data. Time-series (A) needs a sequential time component; image (C) and unstructured text (D) have no fixed column schema.
</details>

### Q4. A team trains a model on a labeled dataset of emails marked as "spam" or "not spam" so the model can predict the label of new emails. Which learning paradigm is this?
- A. Unsupervised learning
- B. Reinforcement learning
- C. Supervised learning
- D. Semi-supervised clustering

<details><summary>Answer & explanation</summary>

**Answer: C**

Learning from labeled data to predict a label is supervised learning. Unsupervised (A) works with unlabeled data (e.g. clustering); reinforcement (B) learns via rewards/interaction with an environment.
</details>

### Q12. A bank's dataset is highly imbalanced (very few fraud cases). The team wants a single metric that balances precision and recall to evaluate the fraud classifier. Which metric should they prioritize, and which is a business metric rather than a model metric?
- A. Prioritize accuracy; ROI is a business metric.
- B. Prioritize F1 score; return on investment (ROI) is a business metric.
- C. Prioritize F1 score; AUC is a business metric.
- D. Prioritize cost per user; accuracy is a business metric.

<details><summary>Answer & explanation</summary>

**Answer: B**

F1 score is the harmonic mean of precision and recall — better suited to imbalanced data than accuracy (accuracy is misleading when the minority class is rare). ROI is a business metric; AUC (C) is a model metric, and cost per user (D) is a business metric, so it is not the choice to "prioritize" as a classifier measure.
</details>

### Q18. A team wants to detect that the statistical distribution of incoming production data has diverged from the training data. What is this phenomenon called?
- A. Overfitting
- B. Data drift
- C. Feature scaling
- D. Hyperparameter tuning

<details><summary>Answer & explanation</summary>

**Answer: B**

A change in the input data distribution relative to the training data is called data drift, which reduces prediction quality. Overfitting (A) is a training-time phenomenon; feature scaling (C) and tuning (D) are processing steps, not drift.
</details>

### Q24. Scenario: A startup must estimate a continuous numeric value — the expected sale price of a house — from features like size, location, and age. Which ML technique and problem type is this?
- A. Classification, because the output is a category
- B. Clustering, because houses form groups
- C. Regression, because the output is a continuous number
- D. Reinforcement learning, because the agent gets rewards

<details><summary>Answer & explanation</summary>

</details>
